# Victory Velocity — Weekly Report Generator

A browser-based tool for building and exporting styled weekly reports for Victory Velocity.

Fill in the form fields (week, highlights, metrics, focus areas, and notes), then use the **Generate Report** button to produce a formatted, print-ready report you can copy or export.

No backend required — runs entirely in the browser.

**Live tool:** https://kadeny-128.github.io/vv-report-generator/
